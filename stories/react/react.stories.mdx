# Handlers

## composeHandlers

<Description of={composeHandlers} />

If the `next` function provided by handler called, it will trigger the next handler, or not.

```tsx
import { composeHandlers } from '@quadrats/react';

const createHandlers = composeHandlers([
  {
    onKeyDown(event, editor, next) {
      // ...
    },
  },
  // ...
]);

// ...

const handlers = createHandlers(editor);

// ...

<Editable {...handlers} />;
```

## composeRenderElements

<Description of={composeRenderElements} />

import { composeRenderElements } from '@quadrats/react';
