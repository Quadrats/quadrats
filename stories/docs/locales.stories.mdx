## Installation

```bash
npm install @quadrats/locales --save

or

yarn add @quadrats/locales
```

## Introduction

This package just only includes `all locales for quadrats` and the interface [LocaleDefinition](https://github.com/Quadrats/quadrats/tree/master/packages/locales/src/typings.ts).
All locales are restricted in the definition and each locales is just an object.

## Usage

Please import each locale since there is no root file.

```ts
import { enUS } from '@quadrats/locales/en-US';
import { zhTW } from '@quadrats/locales/zh-TW';
```
